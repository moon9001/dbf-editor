# DBF 编辑器使用说明

## 环境要求

- **操作系统**：Windows（已测试 Windows Server / Windows 10/11）
- **Python**：3.8+（项目自带虚拟环境，无需系统安装）

---

## 首次安装依赖

如果项目目录下没有 `dbf_editor/venv/` 文件夹，需要先安装依赖。

打开命令提示符（CMD），执行以下命令：

```batch
cd /d F:\AI\xionglaoshi\0610xiong\dbf_editor
C:\Users\Administrator\.workbuddy\binaries\python\versions\3.13.12\python.exe -m venv venv
venv\Scripts\pip install flask dbf openpyxl
```

如果 Python 路径不同，替换为你的 Python 路径即可。依赖说明：

| 包 | 用途 |
|----|------|
| `flask` | Web 服务框架 |
| `dbf` | DBF 文件读写库 |
| `openpyxl` | Excel 导入导出 |

---

## 启动方法

双击 `start_dbf_editor.bat`，浏览器会自动打开。

- **本机访问**：http://127.0.0.1:5000  
- **局域网访问**：启动后控制台会显示局域网 IP，如 http://192.168.x.x:5000（端口固定为 5000）

> 如果 bat 执行后窗口闪退，请先按照上方"首次安装依赖"步骤操作。

## 文件管理

### 上传 DBF
点击标题栏右侧 **📤 上传** 按钮，选择 `.dbf` 文件上传到 `uploads/` 目录。

### 删除上传的 DBF

**方法一**：选中上传文件（📁 标记），标题栏出现 🗑 删除按钮，点击确认删除。

**方法二**：点击工具栏 **📂 文件管理** 按钮，打开文件管理面板：
- 列出所有上传的 DBF 文件，含文件大小
- 当前正在编辑的文件标有"当前"徽章
- 每个文件可单独 **📂 打开** 或 **🗑 删除**
- 删除文件后页面自动刷新

### 下载 DBF
选中文件后点击 **📥 下载**，即可下载原始 DBF 文件。

## 列顺序说明

表格按以下顺序优先排列关键字段（与 VFP 原始顺序一致）：

> **DWDM → DWMC → XM → KASB → ZJH → ZJHM → CSDM → JGDM → CSRQ → MZM → XBM ...**

橙色表头 + 黄色单元格 = 关键信息字段，一眼可辨。

---

## 单值字段说明（DWDM / DWMC）

| 字段 | 含义 | 是否可编辑 |
|------|------|-----------|
| DWDM | 单位代码 | ❌ **只读，不可修改** |
| DWMC | 单位名称 | ❌ **只读，不可修改** |

**原因**：这两个字段是整个 DBF 文件的单位标识，全部记录均相同。若需修改，请直接使用专业 DBF 工具批量替换。

- 双击 DWDM/DWMC 单元格 → 弹出提示"系统固定字段，不可修改"
- 行编辑弹窗中这两个字段显示为灰色只读（标注"只读"标签）
- 新增行时这两个字段自动沿用当前文件的值，无需手动填写

---

## 删除行说明

删除某条记录（某个人）**不会影响** DWDM / DWMC 字段，因为：
- DWDM / DWMC 是每条记录自己保存的字段，删除某行仅删除该行数据
- 其他行的 DWDM / DWMC 不受任何影响
- 删除后文件会自动压缩（pack），编号重新连续

## 导出功能

| 按钮 | 说明 |
|------|------|
| 📊 导出 Excel | 导出为 .xlsx 格式，带样式、冻结首行、自动筛选 |
| 💾 导出 DBF | **直接下载原始 DBF 文件**，格式、编码、字段结构完全保持不变 |

> ⚠️ 导出 DBF 下载的是**当前磁盘上的实际文件**，包含你所有的编辑修改，可以直接在 VFP 或其他工具中使用。

## 常见问题

**Q: 启动后提示端口被占用？**  
A: 请关闭之前打开的 DBF 编辑器窗口，或重启电脑后再试。

**Q: 局域网其他电脑无法访问？**  
A: 检查 Windows 防火墙是否放行端口 5000，或临时关闭防火墙测试。

**Q: 上传的 DBF 中文乱码？**  
A: 工具默认使用 GBK (cp936) 编码读取。如遇乱码，可能是文件编码不同，请用 VFP 重新导出为 GBK 编码。
